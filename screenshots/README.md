# 📸 Screenshots Folder

## 🎯 How to Add Screenshots to README

### 📁 Place Your Screenshots Here
1. **Add your screenshot files** to this folder
2. **Supported formats:** .png, .jpg, .jpeg, .gif
3. **Recommended size:** 1200x800 for desktop, 400x600 for mobile

### 🖼️ Screenshot Names (Recommended)
```
📄 character-selection.png     - Hero selection screen
📄 battle-arena.png        - Battle gameplay
📄 light-mode.png          - Light theme view
📄 dark-mode.png           - Dark theme view
📄 mobile-view.png          - Mobile responsive view
📄 victory-screen.png       - Victory celebration
```

### 🔗 Update README.md Links
After adding screenshots, update the main README.md:

```markdown
## 📸 Screenshots

### 🎮 Main Gameplay
![Character Selection](./screenshots/character-selection.png)
![Battle Arena](./screenshots/battle-arena.png)

### 🎨 Theme System
![Light Mode](./screenshots/light-mode.png)
![Dark Mode](./screenshots/dark-mode.png)

### 📱 Mobile View
![Mobile View](./screenshots/mobile-view.png)
```

### 📱 Upload to GitHub
```bash
git add screenshots/
git commit -m "Add app screenshots"
git push origin main
```
